Un iPad qui affiche « iPad indisponible », « Verrouillage de sécurité » ou demande de le connecter à un ordinateur ne retrouvera pas l’accès grâce à un simple redémarrage. Si un délai est encore affiché, le meilleur réflexe est d’attendre ; si le code est oublié, il faut effacer l’iPad par une méthode Apple officielle, puis restaurer une sauvegarde si elle existe.

## Distinguer un iPad verrouillé d’un iPad réellement bloqué

Le terme « iPad désactivé » recouvre deux situations différentes. Dans la première, l’appareil fonctionne, mais le code d’accès a été saisi plusieurs fois de façon incorrecte. L’écran affiche alors un délai avant le prochain essai, ou un message indiquant que l’iPad est indisponible. C’est une mesure de sécurité normale.

Dans la seconde, l’iPad est aussi figé : écran noir, interface qui ne répond plus ou logo Apple bloqué. Un redémarrage forcé peut alors être utile pour relancer l’appareil. Il ne contournera toutefois ni le code d’accès ni le verrouillage de sécurité.

### Ce que fait — et ne fait pas — le redémarrage forcé

Sur un iPad sans bouton principal, appuyez brièvement sur le bouton de volume le plus proche du bouton supérieur, puis brièvement sur l’autre bouton de volume. Maintenez ensuite le bouton supérieur enfoncé jusqu’à l’apparition du logo Apple.

Sur un iPad doté d’un bouton principal, maintenez simultanément le bouton principal et le bouton supérieur jusqu’au logo Apple.

Cette manipulation est sans danger pour les données : elle force uniquement le redémarrage du système. Au rallumage, l’iPad demandera toujours son code.

:::key[À retenir]
Un redémarrage forcé résout une panne d’affichage ou un iPad figé, pas un oubli de code. Il n’existe aucune combinaison de boutons légitime permettant de supprimer le code d’accès sans effacer l’appareil.
:::

## Vérifier le message affiché avant de réinitialiser l’appareil

Ne lancez pas un effacement trop vite. Le message de l’écran indique généralement la voie la plus simple et permet parfois d’éviter une manipulation inutile.

| Situation affichée ou connue | Action à privilégier | Conséquence pour les données |
|---|---|---|
| Un compte à rebours indique « Réessayer dans… » | Attendre la fin du délai avant une nouvelle saisie, si vous pensez connaître le bon code | Aucune perte de données |
| Le code a été modifié il y a peu et l’option apparaît | Utiliser l’ancien code pour définir un nouveau code | Aucune perte de données |
| « Code oublié ? » ou « Effacer l’iPad » apparaît | Effacer l’iPad directement depuis l’écran verrouillé | Données locales effacées |
| L’iPad demande une connexion à un ordinateur | Passer par le mode de récupération sur Mac ou PC | Données locales effacées |
| L’iPad est accessible dans Localiser | Déclencher un effacement à distance | Données locales effacées une fois l’iPad connecté |

Si un délai est affiché, ne faites pas d’essais au hasard. Une tentative erronée supplémentaire peut rallonger le verrouillage. Prenez le temps de vérifier vos habitudes : code à quatre ou six chiffres, date personnelle, combinaison récemment modifiée, ou confusion avec le code d’un autre appareil.

Sur certains iPad sous une version récente d’iPadOS, si vous avez changé le code au cours des 72 dernières heures, une option permet d’employer l’ancien code pour créer un nouveau code. Elle n’est proposée que dans ce cas précis et seulement si elle s’affiche à l’écran.

:::warning[Attention]
N’essayez pas des séries de codes « probables ». Après plusieurs erreurs, l’iPad peut ne plus proposer de délai et imposer l’effacement. Le code d’accès protège aussi les données chiffrées : Apple ne peut pas le communiquer ni récupérer les données locales à votre place.
:::

## Effacer l’iPad depuis l’écran verrouillé ou avec Localiser

Quand l’option « Code oublié ? », « Effacer l’iPad » ou un libellé voisin s’affiche sur l’écran verrouillé, c’est généralement la méthode la plus directe. Elle est disponible sur les appareils suffisamment récents, avec une version compatible d’iPadOS, déjà connectés à Internet et associés à un compte Apple (ancien identifiant Apple).

### Réinitialiser depuis l’écran de l’iPad

1. Touchez **Code oublié ?** ou **Effacer l’iPad**, selon le texte proposé.
2. Confirmez que vous souhaitez effacer l’appareil.
3. Saisissez le mot de passe du compte Apple associé à l’iPad lorsque le système le demande.
4. L’iPad efface ses réglages et données, puis redémarre sur l’assistant de configuration.
5. Connectez-vous au même compte Apple et choisissez de restaurer une sauvegarde iCloud, si elle est disponible.

Le nom exact des boutons peut varier selon la version d’iPadOS. Si aucune option n’apparaît, n’insistez pas : utilisez un ordinateur en mode de récupération.

### Effacer à distance depuis Localiser

Vous pouvez également utiliser l’app **Localiser** sur un autre iPhone, iPad ou Mac connecté à votre compte, ou ouvrir **iCloud.com/find** dans un navigateur. Sélectionnez l’iPad, puis choisissez l’option d’effacement à distance.

Cette solution est pratique si l’iPad n’est pas sous la main ou si son écran verrouillé ne propose pas l’effacement. L’ordre est exécuté dès que l’appareil se connecte à Internet. S’il est hors ligne, il reste en attente.

Après l’effacement, le verrouillage d’activation demeure actif. Il faudra impérativement renseigner le compte Apple précédemment associé à l’iPad pour l’activer à nouveau. C’est précisément ce qui empêche une personne non autorisée de réutiliser un appareil perdu ou volé.

:::tip[Le conseil UWOS]
Avant d’effacer l’iPad, vérifiez sur un autre appareil que vous connaissez bien le mot de passe de votre compte Apple et que vous pouvez recevoir son éventuelle validation à deux facteurs. C’est le point qui bloque le plus souvent la remise en service après une réinitialisation.
:::

## Utiliser le mode de récupération avec un Mac ou un PC

Le mode de récupération est la solution à privilégier lorsqu’aucune option d’effacement ne s’affiche, que l’iPad n’est pas relié à Internet ou qu’il affiche directement un écran demandant d’être connecté à un ordinateur. Prévoyez un câble USB fiable et, idéalement, branchez l’ordinateur sur secteur.

Sur Mac, ouvrez **Finder**. Sur un PC Windows récent, utilisez l’app **Appareils Apple** ; sur une configuration plus ancienne, iTunes peut encore être nécessaire. Mettez au préalable l’application à jour afin qu’elle puisse télécharger la version d’iPadOS adaptée.

### Mettre l’iPad en mode de récupération

1. Connectez l’iPad à l’ordinateur avec son câble.
2. Sur un iPad sans bouton principal, appuyez brièvement sur le bouton de volume le plus proche du bouton supérieur, puis sur l’autre. Maintenez ensuite le bouton supérieur enfoncé.
3. Ne relâchez pas le bouton lorsque le logo Apple apparaît : continuez jusqu’à voir l’écran du mode de récupération, avec un câble pointant vers un ordinateur.
4. Sur un iPad avec bouton principal, maintenez le bouton principal et le bouton supérieur ensemble jusqu’à cet écran de récupération.
5. Dans Finder, Appareils Apple ou iTunes, sélectionnez l’iPad puis choisissez **Restaurer**.

Le logiciel téléchargera la dernière version compatible d’iPadOS, effacera l’appareil et la réinstallera. Une option « Mettre à jour » peut parfois être proposée : elle cherche normalement à conserver les données, mais ne retire pas un code oublié. Pour résoudre un iPad désactivé faute de code connu, c’est bien **Restaurer** qu’il faut sélectionner.

Si le téléchargement est long et que l’iPad quitte le mode de récupération, laissez le téléchargement se terminer, puis recommencez la séquence de boutons. Évitez de débrancher le câble pendant la restauration, sauf si le logiciel le demande explicitement.

:::stat[En chiffres]
Comptez souvent entre 15 minutes et une heure pour télécharger le système et restaurer l’iPad. La durée dépend de la connexion, du modèle et de l’état du câble ; la récupération d’une grosse sauvegarde iCloud peut ensuite prendre bien davantage.
:::

## Restaurer ses données après l’effacement

Une fois l’iPad redémarré, l’assistant de configuration vous propose plusieurs choix : le configurer comme un nouvel iPad, restaurer une sauvegarde iCloud ou restaurer une sauvegarde depuis un Mac ou un PC. Sélectionnez la sauvegarde la plus récente qui correspond à l’appareil.

Même sans sauvegarde complète, une part importante du contenu peut réapparaître après la connexion au compte Apple : contacts, calendriers, notes, fichiers iCloud Drive, photos synchronisées avec Photos iCloud, applications déjà téléchargées et certains mots de passe enregistrés dans le trousseau iCloud. Cela dépend des réglages de synchronisation qui étaient activés avant le problème.

En revanche, les photos, documents, messages ou données d’applications stockés uniquement sur l’iPad et jamais sauvegardés seront perdus après l’effacement. Le chiffrement du code d’accès empêche toute récupération simple de ces éléments.

:::info[À savoir]
Une synchronisation iCloud n’est pas exactement une sauvegarde iCloud. Par exemple, les photos synchronisées avec iCloud Photos se retéléchargent après connexion au compte, alors qu’une sauvegarde sert surtout à récupérer les réglages, données d’apps et contenus non déjà synchronisés.
:::

## Situations particulières : compte Apple, appareil professionnel et outils tiers

Si vous avez oublié le mot de passe du compte Apple, commencez une récupération de compte via les outils officiels d’Apple. Cette démarche peut prendre du temps selon les informations de sécurité disponibles. Sans ce compte après l’effacement, l’iPad restera bloqué sur le verrouillage d’activation.

Pour un iPad d’entreprise ou d’école, contactez d’abord l’administrateur informatique. L’appareil peut être supervisé par une solution de gestion de parc : l’organisation peut disposer d’une procédure de réinitialisation, de sauvegardes professionnelles ou de restrictions particulières. Effacer l’iPad sans les prévenir peut compliquer sa réinscription dans le parc.

Enfin, méfiez-vous des logiciels et sites qui promettent un « déblocage définitif sans code ni identifiant Apple ». Certains se contentent de déclencher une restauration que vous pouvez faire gratuitement vous-même ; d’autres demandent des paiements, récupèrent des données ou proposent des méthodes non fiables. Ils ne suppriment pas légitimement le verrouillage d’activation.

Si le message affiche encore un délai, attendez et ne tentez le code que si vous êtes certain du bon. Dans tous les autres cas, choisissez la voie officielle la plus adaptée — effacement depuis l’écran, Localiser ou mode de récupération — puis reconnectez-vous à votre compte Apple et restaurez la sauvegarde disponible.
